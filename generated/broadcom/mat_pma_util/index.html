broadcom.mat_pma_util

The MAT Detect collection enables you to detect performance issues and access performance data supplied by the Performance Management Assistant component (PMA) of Mainframe Application Tuner.

Compatible with Zowe CLI Plugin version: 2.0.1

For more information, see the Contents section.

  • get_perf – Get the performance test result for a specific job
